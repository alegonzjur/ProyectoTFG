1) Numpy.

4) Seaborn.

Librera que permite crear grficos estdisticos en Python a partir de conjuntos de datos.

Estos grficos son ms avanzados y suelen utilizarse para crear predicciones basadas en la estadstica.

Entre ellos se encuentra el grfico de regresin

In[]:
sns.lmplot(x='bill_length_mm', y='flipper_length_mm', data=datagit, hue='sex');

O grficos de cajas, que sirven para detectar valores anmalos

In[]:
cols = ['bill_length_mm', 'bill_depth_mm','flipper_length_mm', 'body_mass_g']
i = 0
fig, ax = plt.subplots(ncols=4, figsize=(20,7))
for c in cols:
    sns.boxplot(datagit[c], ax=ax[i])
    i+=1
